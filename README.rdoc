== Cloud9
- Pull code in from git 

- Run bundle 
    bundle install
    this downloads all the gems that we are currently using

- Setup the local mongo database for C9
    mongod --bind_ip=$IP --dbpath=data --rest "$@"

- Run Rails
    rails s -b $IP -p $PORT
